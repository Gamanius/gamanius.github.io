Detailed Description

This namespace holds all function, namespaces, structs and classes that are not directly part of the Renderer but still integrated. A class whose main purpose is to track memory. It will process all allocation done with the MALLOC/FREE define or with the new/delete operator. Note that every allocation will allocate 4 more bytes than needed to save the size.
