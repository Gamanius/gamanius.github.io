The Mouse is moving. Most of the time this message will get posted when the mouse is* inside the client area but if the mouse button has been pressed the mouse move message * can also be posted even when outside window. Possible value is GGeneral::Point<int>

WINDOW_CLOSE 

Close Request of the window. The value returned can be ignored

bool GWindow::init ( )

Will create an dummy OpenGL context and activate it. It will also init all OpenGL calls meaning that this function needs to be called before calling any other rendering functions. If GRenderer::init() this will also be initialized. After activation it will delete itself
