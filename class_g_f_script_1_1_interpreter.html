◆ prepare()

bool GFScript::Interpreter::prepare ( bool  debug = false)

Will take the tokens and create the byte code that can be read and executed by the Interpreter. If the debug bool is set to true the code generated will be slower to execute, but safer. The byte code is also better to debug. If the debug bool is set to false the code generated will be faster but there will not be any safety meaning the Interpreter could crash at all times if bad code has been written.

Parameters
debug- Set to true to enable debug code
Returns
true - If the code generation was successful. False otherwise
