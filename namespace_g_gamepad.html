Detailed Description

Namespace with functions all about game pad inputs. For now this namespace uses the XInput library. Only Controller with XInput support can be recognized by this namespace. There may be support for DirectInput in the future

void GGamepad::enable ( bool  enable)

Will enable/disable all input or output to all gamepads. Gamepad::getState will return a GamepadState struct with all values being 0. This function can be used to disable gamepad controls if the main window looses focus.
